## 1. THE MENTOR – `/mentor`

- Full chat interface.
- Streaming responses via `POST /api/mentor/chat` (SSE).
- Assistant messages with sparkle icon; user messages with avatar.
- Typing indicator while streaming.
- Sidebar: student memory summary (weak topics, learning style) from `GET /api/echo/profile`.
- Quick action chips: "Explain [weak topic]", "Quiz me on [topic]", "Review mistakes", "Simplify concept".
- Conversation history: load previous sessions from `GET /api/mentor/history`.

---

## 2. REVISIT – `/revisit`

- Fetch `GET /api/revisit/plan?type=daily` (default), weekly, sprint tabs.
- Daily: list of tasks with topic, duration, resource links (flashcards, questions).
- Weekly: 7‑day calendar with allocated slots.
- Sprint: intensive plan if exam within 14 days.
- Each task has a "Start" button linking to the resource.

## 3. ASCEND – `/ascend`

- Profile: avatar, level, XP bar, rank badge.
- Subject levels: per‑subject progress bars.
- Recent XP earnings log.
- Leaderboard button → Peak Rankings.
- Unlockables list (themes, frames, badges).
- Data from `GET /api/ascend/profile`.

## 4. SIMVERSE – `/simverse`

- Unified hub with tabs: Geometrix, ForgeField, ReactSphere, BioSphere.
- Each lab must be fully interactive (using existing Three.js / Canvas components, ensure they are functional and not placeholder).
- Geometrix: 3D shape viewer, nets, measurements.
- ForgeField: physics simulations with sliders and graphs.
- ReactSphere: chemistry reactions.
- BioSphere: cell/DNA/ecology explorer.
- PulseLab: experiment evaluation – perform experiment, click "Evaluate", call The Mentor or dedicated endpoint to analyze.
