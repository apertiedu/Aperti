👥 USER ROLES

1️⃣ Admin

Full system control.

Can:

- create/edit/delete teachers
- create/edit/delete assistants
- create/edit/delete admins
- suspend/reactivate accounts
- reset passwords
- assign assistants to teachers
- access ALL data
- access all dashboards
- export all reports
- access logs
- manage backups
- configure system settings

---

2️⃣ Teacher

Can ONLY access their own data.

Can:

- manage students
- create sessions
- create exams/quizzes
- manage assistants
- mark attendance
- enter marks
- access analytics
- export reports
- generate QR codes
- send reports

Cannot:

- access other teachers' data
- manage admins

---

3️⃣ Assistant

Works under a teacher.

Can:

- add students
- mark attendance
- scan QR codes
- enter marks
- view limited reports

Cannot:

- edit sessions
- create exams
- modify analytics structure
- manage assistants
