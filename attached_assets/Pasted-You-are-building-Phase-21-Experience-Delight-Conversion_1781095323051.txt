## 🔥 TOP 5 PRIORITIES (Must be fully implemented)

### 1. Interactive Landing Page Demo & Product Tour
- **Hero Section:** Replace static content with a **live animated dashboard** that auto‑cycles between Teacher, Student, Parent, and Admin dashboards every 4 seconds. Use real UI components (not marketing illustrations). Framer Motion for smooth transitions.
- **Interactive Product Demo:** Add a “Try Aperti” bar with role‑selector buttons. When a visitor clicks “Teacher”, they see a guided walkthrough of creating a course, adding an assessment, and viewing analytics – all using real (mock) data without requiring signup. Implement as a modal overlay with step‑by‑step tooltips.
- **Product Tour:** Create a dedicated `/tour` page that takes a visitor through the entire educational journey (Teacher → Course → Assessment → Student → Parent → Analytics) with animated flow diagrams.
- **Feature Showcase:** Instead of static cards, each feature (Assessment Hub, Question Bank, etc.) opens a mini interactive demo (e.g., a working MCQ, a flip‑able flashcard) directly on the landing page.
- **Academic Command Center Demo:** Add a section that simulates: “Student struggles with Physics” → platform detects weakness → revision pack generated → teacher notified → parent receives report. Show this as a series of animated steps.
- **Smart Calculator:** Add a “Plan Calculator” on the pricing page where teachers input number of students, subjects, courses; the tool recommends the optimal plan and shows monthly cost. Real‑time calculation.

### 2. ContentCraft Upgrade (Educational Notion)
- **Slash Commands:** Implement `/` commands inside the editor for inserting blocks (headings, lists, code, LaTeX, images, etc.).
- **Database View:** Allow creating simple databases (tables) for tracking student progress, question banks, etc., inside a ContentCraft page.
- **Education Blocks:** Add specialized blocks: Learning Objective, Exam Tip, Mark Scheme Reference, Flashcard Embed, Question Embed, Assessment Embed.
- **Nested Pages & Breadcrumbs:** Support sub‑pages with a sidebar tree and breadcrumb navigation.
- **Templates:** Pre‑built templates for lesson plans, revision notes, worksheets, and course outlines.

### 3. Automatic Question Extraction & Classification
- **Upload Interface:** Teacher/admin uploads a past paper (PDF/image).  
- **AI Processing:** Using CoreMind (or Tesseract + rule‑based), extract individual questions, subparts, marks, diagrams.  
- **Classification:** Automatically tag each question with subject, topic, difficulty, command word, paper type, and link to curriculum objectives.  
- **Duplicate Detection:** Compare against existing questions and flag duplicates.  
- **Review Workflow:** Extracted questions go to a moderation queue where teacher/admin can approve, edit, or reject.

### 4. Smart Revision Plan Generator (Revision Center 2.0)
- **Student View:** Student selects “Exam in X days” or “Weak Topics” mode.  
- **Generator:** Combines Echo memory (weak topics), exam dates, teacher priorities, and available flashcards/questions to create a daily revision schedule.  
- **Output:** A calendar with specific tasks each day (review 10 flashcards on Momentum, attempt 5 questions on Electricity, watch a simulation).  
- **One‑Click Generation:** Button “Generate My Revision Plan” on the student dashboard.

### 5. Universal Platform Search (Ctrl+K)
- **Global Search Modal:** Press `Ctrl+K` (or `Cmd+K`) anywhere to open a command‑palette‑style overlay.  
- **Search Scope:** Indexes courses, lessons, questions, assessments, flashcards, revision notes, resources, students (teacher only), reports, settings.  
- **Autocomplete & Quick Actions:** Shows suggestions as you type. Can also perform actions like “Create new assessment”, “Go to attendance”, “Search for momentum questions”.  
- **Backend:** `GET /api/search/universal?q=...` returns categorized results with relevance scores.
